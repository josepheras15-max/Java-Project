# Java project

This is a project I built to practice **Object-Oriented Programming (OOP)**. It shows how different Java classes can connect 

### How it works
*   **The Big Picture:** The `ParkingLot` and `Garage` classes act as the "managers." They hold collections of `Car` objects.
*   **Object Connections:** Each `Car` has its own `Engine`, and each `Garage` has its own `Address`. This taught me how to link multiple classes together.
*   **Data Handling:** I used `ArrayLists` to store multiple cars and practiced 4 different ways to loop through them (For, For-Each, While, and Iterators).
*   **User Input:** I used the `Scanner` class so a user can type in car and engine details directly to add them to the list.
